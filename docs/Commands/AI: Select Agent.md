---
tags: commands
commandName: "AI: Select Agent"
commandSummary: "Opens a picker to select an AI agent for use in the Assistant chat panel."
---

Opens a filter box listing all available agents (both Lua-defined and page-based). The selected agent will be used for subsequent chat sessions in the Assistant panel.

Agents customize the Assistant's behavior through custom system prompts and can optionally restrict which tools are available.

See [[Agents]] for details on creating and configuring agents.
