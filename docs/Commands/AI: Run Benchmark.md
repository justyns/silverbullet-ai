---
tags: commands
commandName: "AI: Run Benchmark"
commandSummary: "Runs benchmark tests against all configured models to measure performance and capabilities."
---

This command navigates to a benchmark page that runs comprehensive tests against all configured models.

Use this to:

- Compare response times across different models
- Verify model capabilities (text generation, embeddings, etc.)
- Identify performance issues or misconfigurations

The results are displayed on the benchmark page with detailed metrics for each model.
