# AICore Library for silverbullet-ai

This library provides Space Lua helpers, widgets, and AI prompt templates for the [silverbullet-ai](https://github.com/justyns/silverbullet-ai) plug.

## Contents

### Space Lua Functions
- `ai.queryLLM(userPrompt, systemPrompt)` - Query the LLM from templates
- `ai.searchEmbeddings(query)` - Search embeddings from templates

### Widgets
- AI Chat On Page - Interactive chat widget

### AI Prompt Templates
Templates for common AI tasks like rewriting, summarizing, and code generation.

### Page Templates
- AI New Chat - Template for creating new chat pages

## Installation

This library is automatically installed when you install the silverbullet-ai plug via the Library Manager.

You can also install it manually:
1. Run `Library: Install`
2. Enter: `github:justyns/silverbullet-ai@main/Library/AICore.md`

Replace `@main` with a specific version tag if needed (e.g., `@0.5.0`).
